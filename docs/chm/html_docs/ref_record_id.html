Record Id

Topic: record_id

Explanation

@record_id:... defines a stable identifier for the current document.

It is useful for contracts, compliance records, internal files, onboarding material, and protocols that need traceable numbering. The value can be referenced via @@ref=meta:record_id.

Examples

@record_id:PROTO-2026-004
