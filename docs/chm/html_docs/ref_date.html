Date

Topic: date

Explanation

Defines the date of the document or meeting.

The parser accepts plain text values such as 2025-06-01 or 24.05.2025. This value is often used in headers, placeholders, filenames, and rendered output.

Examples

@date:2025-06-01
@date:24.05.2025
