Installation And Release Use

Install from npm

npm install -g protoml-parser

Install from source

git clone https://github.com/Ente/protoml-parser.git
cd protoml-parser
npm install -g .

Release structure

  • Source code for development and npm publishing
  • Windows and Linux executables in dist/
  • Native Windows CHM help in docs/chm/ and copied to dist/ during release preparation
  • SHA256SUMS.txt for release checksums

Version check

protoparser --version
