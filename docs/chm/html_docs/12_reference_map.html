Reference Map

This is a map of the language surface, not a full syntax listing.

Metadata and document identity

Common document-level fields include @author, @version, @status, @record_id, @confidentiality, @effective_date, @valid_until, @review_date, and @meta=key:value.

Core structure blocks

Primary content blocks include @participants, @subjects, @tasks, @notes, and @meeting.

Reuse and modularity

ProtoML supports reuse through @import, @macro, @macros_import, @tags_import, and @participants_import.

References and navigation

Reference and navigation features include @@e=id, @@ref=group:id[:field], @@toc, @references, and @attachments.

Governance and approval surface

Formal-document features include @signatures, @approvals, @@signature=id, and @@approval=id.

Analysis and tooling surface

CLI tooling covers rendering, validation, trust inspection, signing and verification, tag statistics, cross-file analysis, register reporting, bundling, macro inspection, scaffolding, and CHM help access.
