Toc

Topic: toc

Explanation

@@toc generates a table of contents inside the @meeting block based on the current meeting headings.

It is useful for longer protocols, contracts, onboarding documents, RCAs, and process documentation.

Optional syntax:

  • @@toc
  • @@toc=2
  • @@toc=3

Examples

@meeting "Minutes"
@@toc
# Overview
## Scope
## Decision
