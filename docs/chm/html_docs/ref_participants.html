Participants

Topic: participants

Explanation

Defines a list of participants for the document.

Each participant uses an ID via =ptX:Name[,alias,email], and can be referenced later using @ptp=ptX or @@e=ptX. If you want to reuse a shared participant list across documents, use @participants_import "participants.pml".

Examples

@participants
=pt1:John Doe,jdoe,jdoe@example.com
=pt2:Jane Doe
